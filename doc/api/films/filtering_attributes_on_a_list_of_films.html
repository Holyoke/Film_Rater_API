Films API

Filtering attributes on a list of films

GET /api/films/

Filtering film resources' attributes by passing in an array of attributes. All attributes are returned by default. Note that params are passed in as an array.

Parameters

Name Description
fields[][year] year of film
fields[][title] title of film
fields[][description] description of film
fields[][url_slug] url_slug of film
fields[][average_rating] average_rating of film

Request

Headers

Accept: application/json
Content-Type: application/json

Route

GET /api/films/

Query Parameters

fields: ["url_slug", "average_rating"]

cURL

curl "localhost:3000/api/films/" -X GET \
	-H "Accept: application/json" \
	-H "Content-Type: application/json"

Status

200 OK
